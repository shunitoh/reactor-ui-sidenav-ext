### Getting Started

``` shell
$ git clone https://github.com/shunitoh/reactor-ui-sidenav-ext.git
$ cd demo
$ npm install
```

### How to Build

```shell
$ gulp build
```

### How to Deploy

```shell
$ node index.js
```

### How to Access

- http://localhost:3333/
